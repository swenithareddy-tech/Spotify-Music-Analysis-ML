# Spotify Music Analysis using Machine Learning

This project analyzes Spotify songs using machine learning techniques.

## Algorithms Used
- KMeans Clustering
- Linear Regression
- Decision Tree
- Random Forest

## Steps Performed
- Data preprocessing
- Feature scaling
- Correlation analysis
- Clustering of songs
- Tempo prediction using regression models
- Model comparison using RMSE

## Technologies
- Python
- numpy
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Output
The models are compared to determine the best algorithm for predicting song tempo.
